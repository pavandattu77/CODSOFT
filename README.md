# CODSOFT Machine Learning Internship

This repository contains the projects completed as part of the CODSOFT Machine Learning Internship.

## Projects

### Task 1: Movie Genre Classification

Built a machine learning model to predict movie genres from plot summaries using Natural Language Processing (NLP).

**Techniques Used**

* Text Cleaning
* TF-IDF Vectorization
* Naive Bayes
* Logistic Regression
* Support Vector Machine (SVM)

---

### Task 2: Credit Card Fraud Detection

Developed a machine learning system to classify transactions as fraudulent or legitimate.

**Techniques Used**

* Data Analysis
* Feature Selection
* Logistic Regression
* Random Forest
* Handling Class Imbalance

---

### Task 3: Customer Churn Prediction

Built a predictive model to determine whether a customer is likely to leave a bank.

**Techniques Used**

* Data Cleaning
* Label Encoding
* Logistic Regression
* Random Forest
* Support Vector Machine (SVM)

---

### Task 4: SMS Spam Detection

Developed an NLP-based system to classify SMS messages as Spam or Ham.

**Techniques Used**

* Text Preprocessing
* Stopword Removal
* TF-IDF Vectorization
* Naive Bayes
* Logistic Regression
* Support Vector Machine (SVM)

---
