# Movie Genre Classification

## Project Overview
This project predicts the genre of a movie based on its plot summary using Machine Learning and Natural Language Processing (NLP).

## Technologies Used
- Python
- Pandas
- Scikit-learn
- NLTK

## Machine Learning Models
- Naive Bayes
- Logistic Regression
- Support Vector Machine (SVM)

## Techniques Used
- Text Cleaning
- TF-IDF Vectorization
- Train-Test Split

## Features
- Dynamic movie plot input
- Genre prediction
- Model comparison


## How to Run

Install dependencies:

```bash
pip install pandas scikit-learn nltk
```

Run project:

```bash
python app.py
```
